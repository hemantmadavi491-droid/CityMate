# 🏙️ CityMate – Smart City Guide & Public Complaint Platform

CityMate is a modern Flask-based web application designed to help citizens access city services, submit public complaints, track complaint status, and explore important facilities across India.

The platform combines a Smart City Guide with a Public Complaint Management System to improve communication between citizens and city authorities.
